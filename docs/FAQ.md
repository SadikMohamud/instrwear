# Frequently Asked Questions

## What is InstrWear?

InstrWear is an on-demand clothing delivery platform that connects users with local merchants.

## How do payments work?

Payments are processed securely using Stripe.

## Can merchants manage products?

Merchant functionality is implemented at the profile level, with product management planned for future expansion.

## Is this a production-ready system?

This project is developed for educational purposes but follows real-world architecture patterns.
